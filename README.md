Z-Player is a retro-styled, offline music player for Android, built with **Jetpack Compose** and **Media3 (ExoPlayer)**. It features a nostalgic cassette tape animation and a modern Material 3 Dark theme.

## üöß Status: Work in Progress

This project is currently under active development. 

### Features Implemented
*   **Retro UI:** Animated Cassette Tape that spins during playback.
*   **Music Library:** Scans local storage for audio files.
*   **Search**: Filter songs by title or artist.
*   **Mini Player:** Persistent playback bar in the library view.
*   **Dynamic Metadata**: Displays real-time audio format details (FLAC/MP3, Sample Rate, Bit Depth).
*   **Swipe Navigation:** Smooth transitions between Player and Library.

## üéØ The Ultimate Goal: Audiophile Grade

Z-Player aims to become the **open-source alternative to high-end players.**

The long-term vision is to implement **Bit-Perfect Audio** by:
*   Bypassing the standard Android Audio Mixer.
*   Communicating **directly with external USB DACs**.
*   Delivering uncompressed, studio-quality sound directly to your ears.
*   Serving the **Audiophile Community** with a transparent, customizable, and high-fidelity engine.
